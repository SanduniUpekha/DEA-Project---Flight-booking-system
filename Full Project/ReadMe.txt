We added full project and following are some details about dashboard.

There are three users
1. Admin
2. Staff G1
3. Staff G2

Admin can manage staff staff and approve staff.
we added admin username and password into 'employee' table. you can log in to admin dashboard using that data.

Staff G1 can manage useres, manage flights and manage tickets.
we added staff G1's username and password to the 'emplyee' table too. you can log in to staff G1 dashboard using that data and see what staff G1 can do.

Staff G2 can manage flights only.
we also added their user name and password to the 'emplyee' table.you can see what staff G2 can do using that data.

Staff cannot log in to the admin dashboard. It will give error message
